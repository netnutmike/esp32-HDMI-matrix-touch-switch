# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-22

### Added
- Initial release of ESP32 CYD HDMI Matrix Touch Switch Controller
- 6-button touch interface in 3x2 grid layout
- Serial communication at 115200 baud (8N1) for HDMI matrix control
- Support for ESP32-2432S028R (Cheap Yellow Display)
- ILI9341 TFT display driver integration
- XPT2046 touch controller support
- Visual feedback with button press animations
- Customizable button labels and commands
- Debug output via USB serial
- Complete PlatformIO project configuration
- Comprehensive README with setup instructions
